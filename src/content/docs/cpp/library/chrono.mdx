C++ includes support for two types of time manipulation:

- The `chrono` library, a flexible collection of types that track time with varying degrees of precision (e.g. <Missing>`std::chrono::time_point`</Missing>).
- C-style date and time library (e.g. <Missing>`std::time`</Missing>).

## Chrono library <Revision since="C++11" noborder />

The `chrono` library defines <Revision until="C++20" noborder>three</Revision><Revision since="C++20" noborder>five</Revision> main types as well as utility functions and common typedefs:
- clocks
- time points
- durations
<RevisionBlock since="C++20" border>
- calendar dates
- time zone information
</RevisionBlock>

## C-style date and time library

Also provided are the C-style date and time functions, such as <Missing>`std::time_t`</Missing>, <Missing>`std::difftime`</Missing>, and <Missing>`CLOCKS_PER_SEC`</Missing>.

## Example

Measures and displays an execution time of a function call.

```cpp
#include <chrono>
#include <iostream>
 
long Fibonacci(unsigned n)
{
    return n < 2 ? n : Fibonacci(n - 1) + Fibonacci(n - 2);
}
 
int main()
{
    const auto start{std::chrono::steady_clock::now()};
    const auto fb{Fibonacci(42)};
    const auto end{std::chrono::steady_clock::now()};
    const std::chrono::duration<double> elapsed_seconds{end - start};
    
    std::cout << "Fibonacci(42): " << fb << "\nElapsed time: ";
//  std::cout << elapsed_seconds.count() << "s\n"; // Before C++20
    std::cout << elapsed_seconds << '\n'; // C++20's chrono::duration operator<<
}
```

Possible output:

```
Fibonacci(42): 267914296
Elapsed time: 0.791429s
```
