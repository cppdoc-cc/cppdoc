
  Returns the smallest possible `year`, that is, `std::chrono::year(-32767)`.
</DeclDoc>

## Return value

`std::chrono::year(-32767)`

## Example

```cpp
#include <chrono>
#include <iostream>

int main()
{
    std::cout << "The minimum year is: " << (int)std::chrono::year::min() << '\n';
}
```

**Output:**

```
The minimum year is: -32767
```
