## Return value

1) `unsigned(x) == unsigned(y)`
2) `unsigned(x) <=> unsigned(y)`

## Example

```cpp
#include <chrono>

int main()
{
    constexpr std::chrono::month m1{6}, m2{8};

    static_assert
    (
        m1 < m2 && m1 == m1 && m1 <= m2 &&
        m2 > m1 && m2 != m1 && m2 >= m1 &&
        m1 <=> m2 != 0
    );
}
```
